Research

Benchmarking AI Models for Automated Code Generation and Testing using HumanEval

MSc Research Project | Jul 2025 - Present | De Montfort University

Presenting a comprehensive evaluation of Large Language Models (LLMs) ‚Äî including CodeT5, SantaCoder, CodeGen, and PolyCoder ‚Äî in automated code generation tasks using the HumanEval benchmark.

Beyond traditional correctness-based metrics, this study examines efficiency, maintainability, and governance, offering a holistic perspective on responsible and sustainable AI adoption in software engineering.

  • Comprehensive pass@k evaluation for code correctness across multiple models
  • Execution time and performance analysis for real-world deployment scenarios
  • Maintainability assessment using automated code quality metrics and linting tools
  • Governance framework evaluation for ethical AI implementation
  • Robustness testing under various input conditions and edge cases

The findings highlight that while models such as CodeT5 demonstrate high accuracy, true reliability also depends on producing code that is efficient, readable, and ethically governed. This research advocates for multi-dimensional benchmarking frameworks to ensure AI-powered coding tools are both effective and trustworthy in professional practice.
